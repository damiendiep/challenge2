Petit guide pour la ligne de commande


Les lignes de commandes essentielles pour utiliser le terminale.



ls La commande ls permet d'afficher le contenu d'un répertoire.

ls -a Afficher les fichiers cachés.

cd Permet de naviguer vers un dossier. ex : "cd home"

cd .. Permet de naviguer vers un dossier parent

pwd Permet de se situer dans les dossiers.

cp Permet de copier un fichier. ex : "cp index.html /home" => copie le fichier index.html vers le dossier "home"

mv / Permet de déplacer un fichier d'un dossier vers un autre. ex : "mv home/index.html ici/"

mv Permet de renommer un fichier. ex : "mv nomdufichier nouveaunomducfichier"

rm Permet de supprimer un fichier. ex : "rm index.html

rm *.txt Permet de supprimer tous les fichiers .txt.

grep Permet de rechercher une chaîne de caractère.
