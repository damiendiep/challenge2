Petit guide pour Git & Github


Personnalisation de l'outil Git


Git config --global user.name "[nom]" S'authentifier à l'outil avec votre nom et prénom

Git config --global user.email "[adresse email]" S'authentifier à l'outil avec votre e-mail


Creation d'un repertoire de dépot


Git init NOMduPROJET Initialisation, cette commande sert à suivre un dossier

Git pull Mettre à jour le dossier local par rapport au serveur Github

Git push Envoie toutes les modifications effectuées

Effectuer des changements


Git status Lister tous les fichiers ajoutés ou modifiés

Git diff Affiche les modifications effectuées

Git add [nomdufichier] Ajoute un fichier à la transaction

Git commit -m "[message]" Valide un fichier "added" et l'enregistre dans les historiques

Git push Envoie toutes les modifications au serveur Github

Git branch Affiche les branches actives

Git branch [nomdelabranche] Creation d'une nouvelle branche

Git checkout [nomdelabranche] S'affecter à une branche

Git merge [nomdelabranche] Combiner les fichiers de la branche choisie, vers la branche où vous êtes situés

Git branch -d [nomdelabranche] Supprimer une branche


Vérifier l'historique des modifications


Git log Affiche l'historique des commits effectués
