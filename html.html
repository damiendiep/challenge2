Petit guide pour HTML & CSS

Les codes nécessaires pour commencer en html & css



La balise "p" Créer un paragraphe ou un texte

La balise "b" Rendre le texte en gras

La balise "a href" Créer un lien

La balise "i" ou "em" Met le texte en italique

La balise "u" Souligne un texte

La balise "strong" Met le texte en gras et rend le mot important pour les moteurs de recherche

La balise "img src" Pour inserer une image ex : "img src="lien de l'image" alt="text""

La balise "ol" Permet de créer une liste ordonnée

La balise "ul" Permet de créer une liste non ordonnée

La balise "li" A utiliser dans les balises "ol" et "ul" pour créer ces élements

La balise "tr" Permet de créer un tableau

La balise "th" A utiliser dans le "tr" pour créer une entête de tableau

La balise "td" A utiliser dans le "tr" pour remplir une colonne

 Le margin padding representent l'espace entre les élèments.
